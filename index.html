Pasos en Google Cloud Console
01console.cloud.google.com → Nuevo proyecto → nombre: bradken-auth
02APIs y servicios → Biblioteca → habilitar Google Sheets API y Google Drive API
03APIs y servicios → Pantalla de consentimiento OAuth → Externo → completar nombre y correo
04Credenciales → + Crear credencial → ID de cliente OAuth 2.0 → Aplicación web
05Orígenes JS autorizados: https://TU-USUARIO.github.io y http://localhost
06Copiar Client ID → pegarlo en "Google Client ID" arriba → Guardar
07Crear Google Sheet con columnas: id nombre cargo curso estado valido_hasta → copiar ID de la URL
08Compartir el Sheet: "Cualquiera con el enlace puede ver" (para que la sincronización pública funcione)
